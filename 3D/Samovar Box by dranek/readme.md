Печатал ABS, потому что считаю его самым проблемным из-за больших термодеформаций после остывания. Лучше всего будет скорей всего из PLA.
Корпус можно распечатать в минимальном виде - Сама коробка (Body.stl) плюс верхняя крышка (Top.stl). Тогда при необходимости блока реле его придется расположить в другом месте, например рядом с регулятором мощности, а с Самоваром соединить семижильным кабелем.
Крышки привинчиваю винтами м3 10мм с головкой в потай, для этого рассверливаю намеченные отверстия до 3 мм. Можно использовать и шурупы, на ваш выбор. Цоколь с корпусом соединяется подобным образом, но у меня нашлось 4 винта потоньше.
В случае исполъзования цоколя нужно в дне корпуса высверлить по намеченным отверстиям и выломать 6 прорезей по краям для вентиляции цоколя и прокладки проводов.
Есть вариант верхней крышки без дисплея, энкодера и кнопки (Top_no LCD.STL).
Вентилятор заложен 50*50*10 мм 12В. Плотно одевается на штырьки верхней крышки с натягом, так что можно дополнительно не привинчивать. Я его подключаю вместо насоса воды. В новом варианте платы добавил возможность запайки гнезд разъёмов для вентилятора на 12 и 5В.
Можно распечатать дополнительно цокольный этаж (Body2.stl) и нижнюю крышку (Bottom.stl), тогда появится место для блока реле, дополнительных разъёмов и др., например блока питания, расширителя портов, драйвера ШД для NEMA23 насоса НБК. На нижней крышке предусмотрено крепление для скобы на колонну. Скоба пластиковая польская, предназначена для труб канализации 50 мм, купил в строймаркете несколько и использую.
Разъёмы я использую GX12, для их установки нужно выбрать удобное место по намеченным мини отверстиям в боковых стенках корпуса и цоколя, убедиться, что разъем не упрётся в другие элементы, рассверлить на нужный диаметр.
Можно использовать другие свои разъёмы по тому же принципу. В случае использования цоколя разъёмы и выключатель рядом с штекером питания предпочтительнее устанавливать в него.
Для разъёмов запаянных на плату тоже самое, выбираем нужные миниотверстия, высверливаем, уголки отламываем, подравниваем надфилем, скальпелем и т.п.
Со стороны модуля c ESP32 предусмотрена возможность вырезания большого прямоугольного выреза. Оно мне нужно для возможности перепрошивки / отладки через USB не изымая микроконтроллер из корпуса. Отключаем основное питание полностью (достаем штекер из разъёма обязательно) подключаем к USB PC, шьем, смотрим COM порт. Через этот же вырез есть возможность снять пинцетом джампер (который можно ведь и под углом 90 гр. запаять). Джампер нужно снимать, если есть необходимость отладки с подключенным основным питанием Самовара. При этом в качестве PC необходимо использовать ноутбук с питанием от батареи либо использовать гальваническую развязку USB. Соответственно нужно будет еще распечатать заглушку для этого выреза (stub.STL).
Можно распечатать логотип с названием (Logo.STL) и приклеить где удобно. Печатать двумя цветами соответственно.
