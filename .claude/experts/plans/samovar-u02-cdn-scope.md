# U-02: удаление невместимого local-Ace контура

## Feature Goal

Оставить текущий CDN editor, удалить не дошедший до runtime local-Ace P1.2 и
сохранить без изменений P1.1 плюс проверенные A/B source manifests.

## Chosen Approach

Production packs не создаются. UI-only generator невозможен без нарушения
P1.1: `Kind.UI` требует SHA реального `ace.pack`. Активный LittleFS равен
`0xC0000`; расчёт `0x120000` был ошибкой незапущенной partition migration.

## Preserve Freeze

- P1.1 code + fixtures;
- `web_src/ui-manifest-v1.json`, `raw-manifest-v1.json`,
  `legacy-route-inventory-v1.json` и три schemas;
- `tools/fixtures/ui_pack_assets_v1/**`;
- `data/edit.htm`, `data/edit.htm.gz`, `partitions/samovar.csv` и
  `platformio.ini`.

## Delete

- `third_party/ace/**`;
- `tools/build_ace_1_36_3.js`;
- `tools/import_ace_1_36_3.py`;
- `tools/build_ui_packs.py`;
- `tools/integration_ui_pack_assets.py`;
- `tools/test_ui_pack_ace_browser.py`;
- `web_src/factory-pack-lock-v1.json`;
- `ui_pack_limits.h`;
- obsolete local-Ace P1.2 plans and generated bytecode for deleted modules.

## Modify

- `tools/smoke_ui_pack_assets.py`: JSON snapshot 12 -> 11, active partition
  check, explicit absence check for deleted contour;
- `tools/static_analysis_sources.json`: remove `ui_pack_limits.h`;
- ADR, master ledger, P1.1 handoff and W2 plan references;
- relevant codebase-map docs and counts.

`tools/ui_pack_assets_contract.py` is preserved. Frozen reserved paths
`/ace.pack` and `/ace/**` remain namespace rules, not runtime artifacts.

## Implementation Order

1. Freeze SHA of preserved scopes.
2. Delete P1.2/local Ace files.
3. Update asset smoke and static inventory.
4. Replace stale plans/ledger claims.
5. Update codebase map.
6. Run targeted/full gates and independent reviewer loop.

## Success Criteria

- local-Ace/P1.2 paths and APIs are absent;
- active partition source resolves to exactly `0xC0000`/786432;
- no active code claims `0x120000`/1179648 or production pack fit;
- asset smoke retains UI=22, required=21, raw=20+9, routes=42,
  negatives=78 and max source-pack size 126976;
- P1.1 reports 6 valid, 65 invalid, 67 runtime and 146 truncation cases;
- preserved freeze SHA are unchanged;
- full smoke, cppcheck, `Samovar`, `Samovar_s3`, buildfs and diff-check pass;
- reviewer returns `PASS: no errors, no warnings`.

## Rollback

Rollback restores only the deleted untracked P1.2 contour. It does not alter
runtime CDN files, partition layout, P1.1 or frozen manifests.
