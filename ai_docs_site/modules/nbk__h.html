nbk

Модуль управления процессом НБК (непрерывная барботажная колонна) обеспечивает автоматизированное управление технологическим процессом: контроль температуры, подачи, мощности, обработку аварий и захлёбов, а также сбор статистики и взаимодействие с пользователем через сообщения. Реализует конечный автомат по этапам программы — разгон, ручной режим, оптимизация, работа.

Ключевые структуры данных
stats — Структура для хранения статистики текущей сессии: средняя скорость подачи, общий объём жидкости, временные метки начала и последнего обновления объёма
nbk_Mo — оптимальная мощность нагрева, вычисленная на этапе оптимизации
nbk_Po — оптимальная подача браги, вычисленная на этапе оптимизации
nbk_opt_in_progress — флаг, указывающий на активность этапа оптимизации
nbk_work_in_pause — флаг паузы в рабочем режиме после захлёба
nbk_M — текущая мощность нагревателя в режиме НБК
nbk_P — текущая скорость подачи жидкости (л/ч)
nbk_Tp — температура пара в парогенераторе
nbk_Tn — целевая температура барды, при которой начинается стабильная работа
nbk_column_inertia — инерционность колонны (в секундах), определяет длительность пауз между изменениями параметров
nbk_opt_next_time — временная метка следующего шага оптимизации (в миллисекундах)
nbk_Tb — текущая температура в кубе (°C)
nbk_dT — допустимое отклонение температуры в кубе
nbk_dD — поправка на давление (°C)
nbk_dP — шаг коррекции подачи (л/ч)
nbk_dM — шаг коррекции мощности (условные единицы)
nbk_work_next_time — время следующего выполнения шага алгоритма (миллисекунды)
nbk_work_pause_stage — этап восстановления после захлёба
ProgramNum — текущий номер исполняемой строки программы
nbk_Mo_temp, nbk_Po_temp — временные значения мощности и подачи, передаваемые из ручной настройки

void set_power(bool On) Включить или выключить питание нагревателя. Аргументы On — true для включения, false для выключения. Возвращает Исключения

void set_current_power(float Volt) Установить текущую мощность нагрева в ваттах. Аргументы Volt — Мощность в ваттах. Возвращает Исключения

void create_data() Создать файл для записи данных текущей сессии. Аргументы Возвращает Исключения

msg_type — Тип сообщения (например, информационное, ошибка, предупреждение). Возвращает Исключения

void set_buzzer(bool fl) Включить или выключить звуковой сигнал (buzzer). Аргументы fl — true для включения, false для выключения. Возвращает Исключения

void set_water_temp(float duty) Установить целевую температуру воды. Аргументы duty — Целевая температура в градусах Цельсия. Возвращает Исключения

index — Индекс требуемого поля (начиная с 0).

direction — Направление (0 — обратно, 1 — вперёд).
target — Целевое количество шагов.
Возвращает true, если команда успешно передана, иначе false. Исключения

Возвращает Расход жидкости в л/ч. Исключения

Возвращает Скорость двигателя в шагах в секунду. Исключения

float i2c_get_liquid_volume() Получить текущий накопленный объём жидкости из датчика. Аргументы Возвращает Объём жидкости в литрах. Исключения

uint16_t get_stepper_speed(void) Получить текущую скорость шагового двигателя. Аргументы Возвращает Текущая скорость в шагах в секунду. Исключения

void nbk_proc() Основной цикл обработки НБК: управление этапами и проверка критических аварий. Аргументы Возвращает Исключения

void handle_nbk_stage_heatup() Обработка этапа разгона (H): нагрев до рабочих температур, контроль перехода к следующему этапу. Аргументы Возвращает Исключения

void handle_nbk_stage_manual() Обработка ручного режима (S): ручная настройка параметров, контроль захлёба. Аргументы Возвращает Исключения

void handle_nbk_stage_optimization() Обработка этапа оптимизации (O): автоматическая подстройка параметров для стабильной работы. Аргументы Возвращает Исключения

void handle_nbk_stage_work() Обработка рабочего режима (W): стабильная работа с регулированием подачи и мощности. Аргументы Возвращает Исключения

bool check_nbk_critical_alarms() Проверить критические аварии: перегрев, недостаток охлаждения и др. Аргументы Возвращает true, если зафиксирована авария и процесс был остановлен. Исключения

void handle_overflow(const String& msg, bool finish, uint32_t pause_ms) Централизованная обработка события захлёба. Аргументы msg — Сообщение для пользователя.
finish — Если true — завершить процесс, иначе поставить на паузу.
pause_ms — Длительность паузы в миллисекундах (если finish == false). Возвращает Исключения

String get_nbk_program() Получить текущую строку программы НБК в виде строки. Аргументы Возвращает Текстовое представление текущего этапа и параметров программы. Исключения

float toPower(float value) Преобразование входного значения (напряжение или мощность) в мощность в ваттах. Аргументы value — входное значение (V или W).

Возвращает Квадратный корень из num; при num < 0 возвращает -1.0f. Исключения

float fromPower(float value) Преобразование мощности в требуемое напряжение питания ТЭНа. Аргументы value — мощность в ваттах.

Исключения Может отправлять предупреждающие сообщения через SendMsg при обнаружении захлёба.
