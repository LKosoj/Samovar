data/style

Модуль предоставляет набор CSS-стилей для визуализации интерфейса веб-приложения, ориентированного на обработку и отображение данных, включая формы загрузки файлов, панели сообщений, историю операций, вкладки и элементы управления. Стили обеспечивают адаптивность, визуальную обратную связь и удобство взаимодействия.

Ключевые структуры данных messages_box — контейнер для отображения системных сообщений с цветовой индикацией уровня важности history_box — плавающее окно с историей действий пользователя tab — компонент переключения между вкладками с поддержкой активного состояния button — Стилизованные кнопки с эффектами наведения и нажатия tabcontent — Контейнеры вкладок, скрытые по умолчанию checkbox, radio — Визуально переопределённые флажки и переключатели tooltip — Элемент с всплывающей подсказкой при наведении prgline — Группа элементов управления с единым стилем для полей ввода, выпадающих списков и меток ingredients — Контейнер для отображения ингредиентов с колонками типа, названия и количества popup — Модальное окно для отображения дополнительной информации или подтверждения действий list-group — Контейнер для вертикального списка элементов с единым стилем
