column_math

Модуль для расчета параметров ректификационной колонны на основе эталонной модели.
Выполняет вычисление рабочих мощностей, потоков и теоретического числа тарелок
с учетом конфигурации колонны и типа сырья. Используется для согласования
технологических параметров с аппаратной частью и логикой управления самогонным аппаратом.

Ключевые структуры данных
ColumnResults — Результаты расчета параметров колонны

struct ColumnResults
Результаты расчета параметров колонны.
Поля
floodPowerW — Мощность захлеба колонны, Вт
workingPowerW — Рабочая мощность, Вт
headsPowerW — Мощность при отборе голов, Вт
bodyEndPowerW — Мощность при завершении отбора тела, Вт
tailsPowerW — Мощность при отборе хвостов, Вт
headsFlowMlH — Поток отбора голов, мл/ч
bodyFlowMaxMlH — Максимальный поток отбора тела, мл/ч
bodyFlowMinMlH — Минимальный поток отбора тела, мл/ч
bodyEndFlowMlH — Поток при завершении отбора тела, мл/ч
tailsFlowMlH — Поток отбора хвостов, мл/ч
maxFlowMlH — Паспортный максимальный поток (при ФЧ=4 на захлебе), мл/ч
theoreticalPlates — Теоретическое число тарелок

ColumnResults calculate_column_etalon(uint8_t rawMaterial)
Рассчитывает параметры колонны по эталонной модели в зависимости от типа сырья.
Аргументы
rawMaterial — Тип сырья: 0=Фрукты, 1=Зерно, 2=Сахар
Возвращает
Структура ColumnResults с рассчитанными параметрами колонны
Исключения
Нет
