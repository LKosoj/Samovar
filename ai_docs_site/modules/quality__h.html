quality

Модуль Quality предназначен для оценки качества процесса дистилляции на основе температурных показателей и стабильности работы самовара. Он анализирует текущие данные с датчика пара, вычисляет метрики качества по нескольким критериям и формирует общую оценку с рекомендациями. Основные показатели включают стабильность температуры, соответствие температуры пару идеальным значениям для этанола и общую оценку качества. Результаты используются для контроля и оптимизации процесса дистилляции.

Ключевые структуры данных
QualityParams — Структура, содержащая оценки качества и рекомендации по улучшению процесса

struct QualityParams
Структура, содержащая оценки качества и рекомендации по улучшению процесса дистилляции
Поля
stabilityScore — Оценка стабильности процесса (0-100)
alcoholScore — Оценка качества спирта (0-100)
tempScore — Оценка температурного режима (0-100)
overallScore — Общая оценка качества (0-100)
recommendation — Рекомендации по улучшению

float calculateStabilityScore()
Оценка стабильности температуры пара на основе исторических данных

Оценка стабильности в диапазоне от 0 до 100
Исключения
Нет описанных исключений

Исключения
