# Глоссарий

## Основные термины

**Аварийная ситуация (Alarm)**  
Критическое состояние системы, требующее немедленного вмешательства: перегрев, отсутствие воды, захлёб. При срабатывании отключается нагрев, включается сигнализация, отправляются уведомления.

**Время прогнозирования (Predictor)**  
Расчёт оставшегося времени дистилляции на основе тренда температуры и спиртуозности. Обновляется каждые 30 секунд. Используется только при значимом изменении параметров (ΔT > 0.01 °C/мин, ΔA > 0.001 %/мин).

**Грейс-период (Grace Period)**  
Временная блокировка детекции примесей после старта или возобновления отбора (30–60 сек). Предотвращает ложные срабатывания на нестабильных участках.

**Детектор примесей (Impurity Detector)**  
Алгоритм, анализирующий тренд температуры в колонне для обнаружения перехода между фракциями. Реагирует на рост температуры ≥0.02 °C за 2 сек. Поддерживает два уровня реакции: коррекция скорости (до 70%) и пауза отбора.

**Захлёб (Flooding)**  
Состояние колонны, при котором скорость подачи превышает пропускную способность насадки. Определяется по резкому росту давления или температуры. Требует снижения мощности и/или подачи.

**Калибровка шагов (Stepper Calibration)**  
Настройка количества шагов двигателя, соответствующих 1 мл жидкости (`StepperStepMl`). Используется для точного дозирования отбора. Значение сохраняется в EEPROM.

**Кольцевой буфер (Circular Buffer)**  
Структура данных для хранения истории значений (температура, давление) с фиксированным размером. При заполнении новые значения перезаписывают старые. Используется в детекторе примесей и анализе стабильности.

**Критическая температура**  
Пороги, при превышении которых срабатывает защита:
- `MAX_WATER_TEMP` — отключение нагрева.
- `MAX_ACP_TEMP` — открытие клапана охлаждения.
- `ALARM_WATER_TEMP` — звуковая сигнализация.

**Мощность нагрева (Power)**  
Управление подачей энергии на ТЭН. Поддерживает два режима:
- Релейный (вкл/выкл).
- ШИМ или регулирование напряжения (при `SAMOVAR_USE_POWER`).

**Насадка (Packing)**  
Наполнитель ректификационной колонны. Плотность (`PackDens`, 0–100%) влияет на эффективность и чувствительность детектора примесей.

**Отбор (Withdrawal)**  
Процесс сбора дистиллята. Может управляться шаговым двигателем (насос) или сервоприводом (клапан). Скорость задаётся в мл/час.

**Пауза (Pause)**  
Остановка отбора без остановки нагрева. Используется при ручном вмешательстве, смене ёмкости или срабатывании детектора.

**Программа отбора (Distillation Program)**  
Последовательность шагов, определяющих режим дистилляции. Каждый шаг задаёт тип перехода (по температуре, спиртуозности, пару), объём, скорость, мощность. Максимум — 30 шагов.

**Режимы работы**
- `DISTILLATION` — простая перегонка.
- `CR` — ректификация.
- `BEER` — пивоварение.
- `NBK` — непрерывная барботажная колонна.

**Сигнал (Signal)**  
Состояние цифрового или аналогового входа/выхода: `HIGH`/`LOW`, `true`/`false`. Используется для управления реле, чтения датчиков.

**Спиртуозность (Alcohol)**  
Крепость дистиллята в %. Определяется по температуре пара с помощью калибровочной таблицы.

**Стабильность (Stability)**  
Мера постоянства температуры пара. Оценивается по стандартному отклонению за 60 сек. Низкая стабильность снижает общую оценку качества.

**Температурный тренд**  
Скорость изменения температуры (°C/мин), рассчитываемая методом наименьших квадратов. Используется в прогнозировании и детекции примесей.

**Точка росы**  
Температура, при которой пар начинает конденсироваться. Контролируется для предотвращения захлёба.

**Уставка (Setpoint)**  
Целевое значение параметра: температура, мощность, объём. Система стремится к достижению и поддержанию уставки.

**Фракция**  
Часть дистиллята с определёнными свойствами:
- Головы — легкие фракции.
- Тело — основной продукт.
- Хвосты — тяжёлые фракции.

**Шаговый двигатель (Stepper)**  
Исполнительный механизм для точного дозирования отбора. Управляется по шагам. Поддерживает плавное ускорение/замедление.

**Эталонная модель (Etalon)**  
Расчётные параметры колонны (мощность, потоки) на основе физических характеристик: диаметр, высота, тип сырья. Используется для автоматической настройки режимов.
