# debug_custom.json

Конфигурационный файл определяет параметры отладки для проекта на базе платформы Arduino, работающего на микроконтроллере ESP32. Он задаёт инструментарий для отладки, путь к файлу описания периферии (SVD), а также команды, выполняемые при подключении и перезапуске отладки. Файл используется, например, в средах с поддержкой OpenOCD или J-Link для настройки корректного взаимодействия с целевым устройством.

Секции и ключи

name — отображаемое имя конфигурации, связанное с целевой платформой.<br>
toolchainPrefix — префикс используемого компилятора и инструментов для кросс-компиляции, в данном случае для архитектуры Xtensa (ESP32).<br>
svdFile — путь к файлу SVD (System View Description), содержащему описание регистров и периферийных блоков микроконтроллера.<br>
request — тип запрашиваемой отладочной операции; значение "attach" означает подключение к уже запущенному устройству.<br>
postAttachCommands — список команд отладчика (например, GDB), выполняемых сразу после подключения к устройству.<br>
overrideRestartCommands — команды, которые выполняются при перезапуске сессии отладки вместо стандартных действий.

Важные параметры

postAttachCommands — используются для инициализации устройства при подключении, включая остановку после сброса и установку начальной точки останова.<br>
overrideRestartCommands — обеспечивают согласованный способ перезапуска отладки, повторяя ключевые шаги инициализации.
