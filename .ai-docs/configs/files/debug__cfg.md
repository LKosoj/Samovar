# debug.cfg

Конфигурационный файл предназначен для настройки OpenOCD при отладке платы ESP32-WROVER-KIT. Он определяет интерфейс подключения JTAG, используемый адаптер, напряжение питания флеш-памяти и загружает конфигурацию целевого устройства ESP32. Файл используется при запуске OpenOCD и обеспечивает корректное взаимодействие с отладочным оборудованием.

Секции и ключи

source — включает в конфигурацию содержимое другого конфигурационного файла; используется для подключения драйвера интерфейса и конфигурации целевого чипа<br>
set ESP32_FLASH_VOLTAGE 3.3 — устанавливает напряжение питания флеш-памяти в вольтах, необходимое для корректной работы внешней флеш-памяти на плате

Важные параметры

ESP32_FLASH_VOLTAGE — задаёт уровень напряжения (в вольтах), подаваемого на флеш-память; для ESP32-WROVER-KIT обычно устанавливается в 3.3 В, что соответствует логическому уровню платы
