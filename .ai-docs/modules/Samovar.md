# Samovar

/**
 * @brief Модуль управления процессами самогонного аппарата на базе ESP32.
 *
 * Обеспечивает хранение и обмен состоянием между компонентами системы: датчиками,
 * исполнительными устройствами, веб-интерфейсом и внешними сервисами (MQTT, облако).
 * Содержит глобальные переменные, управляющие параметрами процесса, и функции остановки
 * и уведомления. Поддерживает расширенные режимы контроля (давление, мощность, теплопотери).
 *
 * Ключевые структуры данных
 * MESSAGE_TYPE — Перечисление уровней сообщений (не определено явно, но используется через `msg_level`)
 */

---
void stop_process(String reason)
Останавливает текущий технологический процесс с указанием причины.
Аргументы
reason — Текстовое описание причины остановки процесса
Возвращает
Нет
Исключения
Нет
