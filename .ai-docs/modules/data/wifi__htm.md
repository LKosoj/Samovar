# data/wifi

```markdown
Краткий веб-интерфейс для настройки Wi-Fi на устройстве Samovar.  
Позволяет сохранить или удалить учётные данные сети Wi-Fi.  
После любого действия устройство перезагружается для применения изменений.

Ключевые структуры данных  
WiFiConfig — Данные для подключения к Wi-Fi: SSID и пароль

---
POST /wifi/save  
Сохраняет учётные данные Wi-Fi и инициирует перезагрузку  
Аргументы  
ssid — Имя сети Wi-Fi (до 63 символов)  
pass — Пароль сети (до 63 символов, может быть пустым)  
Возвращает  
Редирект на главную страницу после сохранения и перезагрузки  
Исключения  
HTTP 400 — Если SSID пустой или превышает длину

---
POST /wifi/clear  
Стирает сохранённые учётные данные Wi-Fi и инициирует перезагрузку  
Возвращает  
Редирект на главную страницу после очистки и перезагрузки
```
