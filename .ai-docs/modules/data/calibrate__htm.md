# data/calibrate

/**
 * @brief Веб-интерфейс для калибровки шагового двигателя насоса.
 *
 * Модуль предоставляет HTML-страницу с интерактивным интерфейсом для калибровки шагового двигателя дозирующего насоса.
 * Поддерживается выбор типа насоса (встроенный или внешний I2C), настройка скорости двигателя и выполнение калибровки.
 * Интерфейс взаимодействует с сервером через HTTP-запросы для запуска и завершения процесса калибровки.
 *
 * Ключевые структуры данных
 * stepperStepMlLocal — Количество шагов двигателя на 100 мл для встроенного насоса (локальное значение).
 * stepperStepMlI2C — Количество шагов двигателя на 100 мл для внешнего I2C-насоса.
 */

---
getPumpType()
Возвращает текущий тип выбранного насоса.
Аргументы
Возвращает
Тип насоса: "local" или "i2c".
Исключения

---
onPumpTypeChange()
Обновляет отображаемое значение шагов на 100 мл в зависимости от выбранного типа насоса.
Аргументы
Возвращает
Исключения

---
calibrate()
Запускает или завершает процесс калибровки шагового двигателя.
Аргументы
Возвращает
Всегда возвращает 0.
Исключения
Диалог alert при ошибке HTTP-запроса.

---
window.onload()
Инициализирует интерфейс при загрузке страницы.
Аргументы
Возвращает
Исключения
