# partition_manager

Модуль предназначен для интеграции инструмента управления разделами ESP32 в процесс сборки PlatformIO.  
Он добавляет путь к утилите `esp32partitiontool` в переменную окружения Python и загружает менеджер партиций.  
Используется на этапе построения прошивки для корректной обработки конфигурации flash-памяти.

Ключевые структуры данных  
PartitionManager — объект, управляющий конфигурацией разделов flash-памяти ESP32

---
load_pm(env)  
Загружает и инициализирует менеджер разделов для окружения сборки ESP32  
Аргументы  
env — объект окружения PlatformIO, содержащий параметры сборки и платформу  
Возвращает  
Инициализированный экземпляр PartitionManager  
Исключения  
Вызывает исключение при отсутствии необходимых путей или некорректной конфигурации платформы
