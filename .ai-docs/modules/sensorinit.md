# sensorinit

Модуль `Samovar_S_I` предоставляет функциональность для управления самоваром, включая загрузку программ в зависимости от режима работы, обработку данных с датчиков температуры, давления и влажности, взаимодействие с периферийными устройствами (насос, ШИМ, датчики по 1-Wire и I2C), а также поддержку Lua-скриптов при соответствующей конфигурации. Модуль обеспечивает синхронизацию доступа к шине I2C с использованием семафоров и реализует коррекцию температурных показаний на основе атмосферного давления. Включает функции инициализации, опроса датчиков, управления режимами работы, формирования JSON-ответов и логирования данных.

Ключевые структуры данных  
DeviceAddress — массив из 8 байт, используемый библиотекой DallasTemperature для хранения уникального адреса датчика температуры DS18B20  
SamSetup — глобальная структура, содержащая поправки температур для каждого датчика (DeltaSteamTemp, DeltaPipeTemp и др.)  
SteamSensor, PipeSensor, WaterSensor, TankSensor, ACPSensor — структуры типа SensorData, хранящие текущую, предыдущую температуру и счётчик ошибок  
DSAddr — глобальный массив, содержащий адреса устройств 1-Wire
