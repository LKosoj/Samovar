# partition_manager

```markdown
Модуль обеспечивает загрузку менеджера партиций (Partition Manager) для ESP32 в рамках сборки проекта в PlatformIO.  
Он добавляет путь к утилите `esp32partitiontool` в переменную окружения Python и импортирует необходимые компоненты.  
Затем вызывает функцию `load_pm`, инициируя обработку конфигурации разделов прошивки.

Ключевые структуры данных  
PartitionManager — объект, управляющий конфигурацией и размещением разделов в памяти устройства ESP32

---
load_pm(env)  
Инициализирует и загружает менеджер партиций для окружения сборки ESP32  
Аргументы  
env — объект окружения PlatformIO, содержащий параметры сборки и платформы  
Возвращает  
None  
Исключения  
Может вызвать ImportError, если esp32partitiontool не найден или не загружен
```
