# nbk

Модуль управления процессом НБК (непрерывная барботажная колонна) обеспечивает автоматизированное управление технологическим процессом: контроль температуры, подачи, мощности, обработку аварий и захлёбов, а также сбор статистики и взаимодействие с пользователем через сообщения. Реализует конечный автомат по этапам программы — разгон, ручной режим, оптимизация, работа.

Ключевые структуры данных  
stats — Структура для хранения статистики текущей сессии: средняя скорость подачи, общий объём жидкости, временные метки начала и последнего обновления объёма  
nbk_Mo — оптимальная мощность нагрева, вычисленная на этапе оптимизации  
nbk_Po — оптимальная подача браги, вычисленная на этапе оптимизации  
nbk_opt_in_progress — флаг, указывающий на активность этапа оптимизации  
nbk_work_in_pause — флаг паузы в рабочем режиме после захлёба  
nbk_M — текущая мощность нагревателя в режиме НБК  
nbk_P — текущая скорость подачи жидкости (л/ч)  
nbk_Tp — температура пара в парогенераторе  
nbk_Tn — целевая температура барды, при которой начинается стабильная работа  
nbk_column_inertia — инерционность колонны (в секундах), определяет длительность пауз между изменениями параметров  
nbk_opt_next_time — временная метка следующего шага оптимизации (в миллисекундах)  
nbk_Tb — текущая температура в кубе (°C)  
nbk_dT — допустимое отклонение температуры в кубе  
nbk_dD — поправка на давление (°C)  
nbk_dP — шаг коррекции подачи (л/ч)  
nbk_dM — шаг коррекции мощности (условные единицы)  
nbk_work_next_time — время следующего выполнения шага алгоритма (миллисекунды)  
nbk_work_pause_stage — этап восстановления после захлёба  
ProgramNum — текущий номер исполняемой строки программы  
nbk_Mo_temp, nbk_Po_temp — временные значения мощности и подачи, передаваемые из ручной настройки
