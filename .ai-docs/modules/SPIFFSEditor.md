# SPIFFSEditor

```markdown
Модуль предоставляет веб-интерфейс для редактирования файлов в файловой системе SPIFFS (или другой, совместимой с `fs::FS`) на устройствах ESP8266/ESP32 через асинхронный веб-сервер. Он позволяет просматривать содержимое файловой системы, редактировать, загружать и удалять файлы, а также поддерживает исключение определённых файлов или путей из списка через файл `.exclude.files`. Реализован как обработчик запросов `AsyncWebHandler`, интегрируемый в `ESPAsyncWebServer`.

Ключевые структуры данных  
ExcludeList — односвязный список для хранения шаблонов имён файлов, которые должны быть исключены из отображения в редакторе  
_fs — Экземпляр файловой системы (например, SPIFFS), используемый для операций чтения/записи файлов  
request->_tempFile — Временный файл, создаваемый во время загрузки, хранится в объекте запроса

class SPIFFSEditor
Асинхронный веб-обработчик для управления файлами в SPIFFS через веб-интерфейс
Поля
_fs — ссылка на экземпляр файловой системы (FS), с которой работает редактор

SPIFFSEditor(const fs::FS& fs) — конструктор, инициализирует файловую систему
bool canHandle(AsyncWebServerRequest *request) const — проверяет, может ли обработчик взять на себя обработку запроса
```
