Установка параметров работы Самовара

 
Автоматика “Самовар” поддерживает  несколько режимов работы:
 
  • Режим первичной дистилляции.
     
  • Режим ректификации
  •  Отбор голов;
  •  Отбор тела;
  •  Работа в предзахлебном режиме 
  •  Отбор хвостовых  фракций  (ароматных вод)
  •  Работа в режиме паузы 
 
3. Работа в ручном режиме
 
4. Варка пива 
 
5. Су-Вид (Sous Vide)
 
 
Переключение между режимами.
 
При включении Самовар загружается в последнем  использованном режиме.
Для переключения на другой режим нужно:
 
1. Перейти в меню “Настройки”
 
 
 
2. На вкладке "Основные", в выпадающем меню выбрать нужный режим.
 
 
 
                       
3. Сохранить изменения.
 
 
 Затем нужно перезагрузить Самовар и он загрузится в выбранном режиме работы. Также нужно обновить окно браузера.
 
    Внимание! При переключении режимов, после нажатия кнопки “Сохранить” Самовар сохранит  только те датчики температуры, которые были подключены к нему в данный момент.  Впоследствии, когда после дистилляции вы соберете колонну для ректификации, вновь подключенные датчики  температуры определятся Самоваром , но температуры отображаться не будут.  Вместо  реальной температуры будет значение 0,000.
Чтобы исправить,  нужно будет пройти по пути Настройки - Температура и в выпадающих меню каждого датчика выбрать актуальный. 
 
 
Этого  можно не делать каждый раз при смене режимов, если сохранить существующие настройки.
Для  сохранения настоек в меню "Настройки" выберите вкладку "Прочие" и  нажмите “Сохранить настройки”(1).
 
 
 
 
 
Настройки в виде текстового файла  samovarbackup.txt сохранятся на ваш компьютер.
 
Файл можно переименовать (например, название режима) и потом загружать (2) в нужной конфигурации оборудования.
